# 객체지향의 사실과 오해

## 0장

### 서문

이 책은 객체지향이란 무엇인가라는 원론적이면서도 다소 위험스러운 질문에 대한 저자 나름의 대답을 말하기 위해 쓰여짐

객체지향으로 향하는 걸음

1. 클래스가 아니라 객체를 바라보는 것
2. 객체를 독립적인 존재가 아니라 기능을 구현하기 위해 협력하는 공동체의 존재로 바라보는 것
3. 협력에 참여하는 객체들에게 얼마나 적절한 역할과 책임을 부여할 수 있느냐
4. 앞서 설명한 개념들을 여러분이 사용하는 프로그래밍 언어라는 틀에 흐트러짐 없이 담아낼 수 있는 기술을 익히는 것

이 책의 목적은 객체지향의 첫 번째 걸음에서 마지막 걸음까지 재빠르게 내디딜 수 있도록 도움을 주는 것

책을 덮은 후 여러분들이 지금까지 해왔던 방식을 뒤돌아보면서 깊은 사색에 잠길 수만 있다면 이 책은 그 목적을 다한 것이라고 생각함

### 대상독자

객체지향 프로그래밍 언어를 사용하는 소프트웨어 개발자

이 책의 목표는 페이지를 한장 한장 넘길 때마다 프로그래밍을 했던 기억을 떠올리며 “그게 이런 의미였구나”, “그래서 그 언어가 그런 방식으로 설계됐구나”라는 생각을 하도록 만드는 것

객체지향 프로그래밍 언어에 대한 지식이 많으면 많을수록 책의 행간에 숨겨져 있는 의미를 좀 더 깊이 이해할 수 있을 것

### 책의 구성

1. 협력하는 객체들의 공동체
    - 객체지향 패러다임의 핵심이 자율적인 객체들의 협력이라는 사실을 강조
    - 지금까지 객체지향의 중심이 클래스나 상속이라고 생각했던 사람들은 1장을 읽고 약간의 당혹감을 느낄 수도 있음
2. 이상한 나라의 객체
    - ‘객체란 무엇인가’라는 본질적인 질문에 대답하려고 함
    - 루이스 캐롤이 쓴 ‘이상한 나라의 앨리스’를 모티브로 함
    - 앨리스의 이야기를 듣고 나면 객체가 상태와 행동, 식별자를 가진 존재라는 사실을 이해하게 될 것
3. 타입과 추상화
    - 소프트웨어 개발에서 가장 중요한 개념인 추상화와 마주치게 될 것
    - 추상화는 단순화를 의미
    - 객체지향 패러다임에서 가장 널리 사용ㄱ되는 추상화는 동적인 객체들을 단순화시켜 정적인 타입으로 갈무리하는 것
    - 타입의 개념을 이해하고 나면 객체를 구현하기 위해 클래스를 사용하는 이유를 이해하게 될 것
4. 역할, 책임, 협력
    - 객체지향 설계의 가장 중요한 재료인 역할, 책임, 협력에 관해 설명
    - 객체들은 협력에 참여하기 위해 특정한 역할을 맡고 역할에 적합한 책임을 수행하게 됨
    - 대부분의 사람들은 객체지향이라는 말에서 클래스와 상속을 떠올리겠지만 4장을 읽고 나면 역할, 책임, 협력이 객체지향의 핵심이라는 사실을 알게될 것
5. 책임과 메시지
    - 훌륭한 메시지가 훌륭한 객체지향 설계의 기반이라는 사실 강조
    - 객체의 자율성과 설계의 유연성은 얼마나 훌륭한 메시지를 선택하느냐에 달려 있음
    - 가장 중요한 장
6. 객체 지도
    - 객체지향이 구조와 기능이라는 두 가지 관점을 어떻게 조화시키는지에 관해 설명
    - 구조는 기능에 비해 변화에 더 안정적임
    - 따라서 객체지향 패러다임은 객체들의 구조 안에 기능을 녹임으로써 변화에 안정적인 소프트웨어를 개발할 수 있게 함
7. 함께 모으기
    - 처음이자 마지막으로 구현 코드 동작
    - 그동안 다룬 내용을 동작하는 코드를 통해 정리하는 것이 목표
    - 추상적으로 흩어져 있던 개념들이 좀 더 명확하고 구체적인 실체로 다가올 것

### 책을 읽는 방법

1장부터 7장까지 유기적인 흐름으로 연결돼 있음

이 책을 읽는 가장 효과적인 방법은 처음부터 끝까지 차례대로 읽어 내려가는 것

**이 책이 세상에 나오게 된 동기**

지금 집필중인 다른 책을 이해하는 데 필요한 개념과 배경 지식을 제공하고 싶었기 때문

**이 이야기 말하는 이유**

집필 의도를 명확하게 알고 책을 보는 것이 중요하기 때문

객체지향 프로그래밍에 대한 구체적이고 상세한 지식 전달 X

객체지향을 다루는 다른 책이나 자료들을 읽을 때 미리 알고 있으면 도움될만한 기본 배경과 지식을 다루는 책

이 책을 읽으면 객체지향을 바라보는 시야가 깊어짐과 동시에 넓어질 것
